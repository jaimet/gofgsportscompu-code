This software uses location services to track your current position. It is used to track your exercise and provide statistical information like distance, average speed and so on. Tracked data is stored locally on your device using files ('Tracks'). This functionality can not be disabled, since location services form a core functionality of this software.
The tracked data is:
  • Time
  • Position (longitude, latitude)
  • Altitude
  • Speed
  • Accuracy
A button for deleting recorded data can be found on the details page of each track (reachable via the 'Tracks' page).
If you decide to upload a track to gofg.at the following information will be uploaded:
  • start- & end-time
  • total distance
  • elevation gain & loss
  • maximum speed
